# 글레이드

&#x20;글레이드는 GTK 툴킷과 그놈 데스크톱 환경을 위한 프로그램의 UI 제작 도구입니다.

XML 형식으로 저장되며 C, C++, C#, 발라, 자바, 펄, 파이썬 등 많은 프로그래밍 언어에 붙여서 사용될 수 있습니다.

## 설치

터미널을 열어 다음과 같이 입력합니다.

```
sudo apt install glade
```

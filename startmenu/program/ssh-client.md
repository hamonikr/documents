# SSH Client

SSH은 네트워크 상의 다른 컴퓨터에 로그인하거나 원격 시스템에서 명령을 실행하고 다른 시스템으로 파일을 복사할 수 있도록 해주는 응용 프로그램 또는 그 프로토콜을 말합니다.&#x20;

SSH Client는 SSH 프로토콜을 통해 명령을 내리는 도구입니다.&#x20;
